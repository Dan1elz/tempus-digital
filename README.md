# Tempus Digital

Este README irá guiá-lo para baixar, instalar as dependências e rodar o projeto Tempus Digital.

## Pré-requisitos

- [Node.js](https://nodejs.org/) (recomendado: versão LTS)
- [Angular CLI](https://angular.io/cli)
- [Git](https://git-scm.com/)

## Passos para rodar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Dan1elz/tempus-digital.git
cd tempus-digital
```

### 2. Instale o Angular CLI (caso não tenha)

```bash
npm install -g @angular/cli
```

### 3. Instale as dependências do projeto

```bash
npm install
```

### 4. Rode o projeto localmente

```bash
ng serve
```

Acesse [http://localhost:4200](http://localhost:4200) no seu navegador para visualizar o projeto.

## Observações

- Certifique-se de que o Node.js está corretamente instalado (`node -v`).
- Caso encontre problemas, apague a pasta `node_modules` e rode `npm install` novamente.
